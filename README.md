# Badness enumiration не работает

Перечесление всех российских приложений не работает. В какой-то момент появится приложение которого нет в списке, вы не успеете обновить конфиг и вся ваша фильтрация не работает.

Вместо этого нужно перечислять приложения которым разрешено ходить через впн.

## Структура

```
foreign-android-apps/
└── categories/ # вручную набитые категории приложений с именами пакетов
└── all-packages.yml # генерируется автоматически при каждом изменении в папке категорий
```

## Как помочь?

Создавайте пул реквесты, добавляйте имена пакетов, которые могут идти в впн. В пул реквесте прикладывайте ссылку на приложение(гитхаб, сайт, гугл плэй). Приоритет будет отдаваться приложениям недоступным в РФ. Все российские приложения будут отклоняться. 

### Как узнать имя пакета? 

Откройте страницу любого приложения в Google Play (например хром: `https://play.google.com/store/apps/details?id=com.android.chrome`)

Имя пакета это всё что идет после `id=`, тоесть `com.android.chrome`.

Если приложения нет в сторе, то имя пакета можно глянуть в настройках телефона.
